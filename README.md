# Страница-резюме

Репозиторий содержит исходный код для персональной страницы-резюме. Проект разработан для демонстрации профессионального профиля, навыков и портфолио в чистом и современном формате.

## Особенности

- Адаптивный дизайн для различных устройств
- Разделы для профиля, навыков, опыта, образования и портфолио
- Оформление в стиле рабочего окружения тайлингового оконного менеджера
- Имитирует эстетику текстовых редакторов

## Технический стек

* **Framework:** Angular 21.
* **Styles:** LESS + CSS Custom Properties.
* **Graphics:** WebGL (GLSL Shaders) для высокопроизводительных фильтров.
* **DevOps:** Nginx.

## Начало работы

### Установка

1. Клонируйте репозиторий:
  ```bash
  git clone https://github.com/MrDvD/cv-landing-frontend.git
  ```
2. Перейдите в директорию проекта:
  ```bash
  cd cv-landing
  ```

### Запуск

Для запуска проекта используйте следующие команды:
```bash
sudo docker compose down &&
sudo docker compose build &&
sudo docker compose up
```
